# AWS Serverless Hospital Lab

Projeto de laboratório desenvolvido na AWS utilizando arquitetura serverless para processamento automático de dados hospitalares.

---

## Arquitetura do projeto

Fluxo da aplicação:

1. Arquivo enviado para o Amazon S3
2. Evento do S3 aciona uma função AWS Lambda
3. A Lambda processa os dados
4. Informações são armazenadas automaticamente no Amazon DynamoDB

---

## Serviços AWS utilizados

- Amazon S3
- AWS Lambda
- Amazon DynamoDB

---

## Objetivo do projeto

Simular um fluxo automatizado de processamento de dados hospitalares utilizando serviços serverless da AWS.

---

## Conceitos praticados

- Arquitetura Serverless
- Processamento orientado a eventos
- Integração entre serviços AWS
- Automação em nuvem
- Banco de dados NoSQL
---

## Fluxo da automação

S3 → Lambda → DynamoDB
